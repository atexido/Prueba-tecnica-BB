Ejecución:

Ejecutar el script main.py para correr el código


Proceso:

El proceso lo compuse en 3 scripts, uno para películas, otro para series y el ultimo para las temporadas y capítulos los cuales están en la carpeta scripts. 

La base de datos se guarda en 3 archivos diferentes dentro de la carpeta database, lo óptimo sería crear una base de datos general y dentro de ella 3 tablas con sus respectivas primary key y foreign key pero la hice con sqlite3 para que sea de acceso local y no hayan problemas de compatibilidad

Todos los códigos los comenté para que se entienda el paso a paso.
